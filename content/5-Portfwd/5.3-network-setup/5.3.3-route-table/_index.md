1. Open the **Amazon RDS (Aurora) Console**.
![Networking Session](/AWS_Intern-Report/images/picture22.png)

2. Choose **Subnet groups** from the navigation pane and click **Create DB subnet group**.
![Networking Session](/AWS_Intern-Report/images/picture23.png)
![Networking Session](/AWS_Intern-Report/images/picture24.png)

3. **Name:** Enter a name for your DB subnet group.

4. **VPC:** Choose the VPC you created earlier.
![Networking Session](/AWS_Intern-Report/images/picture25.png)

5. Click **Create** (or Create subnet).
