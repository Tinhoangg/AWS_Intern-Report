1. Open the **EC2 Console**.
![Networking Session](/AWS_Intern-Report/images/picture17.png)

2. Choose **Security Groups** from the navigation pane and click **Create security group**.
![Networking Session](/AWS_Intern-Report/images/picture18.png)
![Networking Session](/AWS_Intern-Report/images/picture19.png)

3. **Security group name:** Enter a name for your security group.

4. **VPC:** Choose the VPC you created earlier.

5. **Inbound rules:** Create the inbound rules as shown below.
![Networking Session](/AWS_Intern-Report/images/picture20.png)

6. Repeat steps 2-5 to create another security group if needed.
![Networking Session](/AWS_Intern-Report/images/picture21.png)
